# 8ppoi SDK

8ppoi（はちっぽい）は、ブラウザゲームの開発・実行環境です。
誰でも簡単にカートリッジが作れます。

公式サイト:
[https://8ppoi.com/](https://8ppoi.com/)
SDKインストール:
[https://8ppoi.com/sdk/](https://8ppoi.com/sdk/)

## 概要

- 入力: 仮想ゲームパッド（4方向 + 汎用ボタン4個）2系統
- オーディオ: 矩形波チャンネル、半音単位周波数、複数同時再生可
- ビデオ: 任意解像度、16色固定パレット + 透明、文字列/配列でグラフィック生成
- コンソール: 60fps固定、カートリッジ連携、リセット・フレーム毎メソッド呼び出し
